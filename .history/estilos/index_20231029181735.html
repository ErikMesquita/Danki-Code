Estilos?

Ainda bem que eu já estou ciente sobre isso, caso contrário ficaria perdido.
O estilo retradado aqui no HTML, não passa das divisões de enquadramento das caixas onde ficam os documentos escritos ou inclementados de forma que fiquem organizados semânticamente.
E o que seria essa semântica? Nada mais é do que o significado para cada elemento, é como se cada elemento tivesse, e tem, uma função própria, pois existem, por algumas vezes duas ou mais formas de codar um projeto interno no HTML, no entanto só existe um caminho certo, que é usar as TAGS semânticas, ou sejam, as tags que tem significado para executar tal ação.

Bora para o conteúdo

style

Usado para adicionar um estilo CSS ao próprio documento.
Obs: Usa-se no head

div

Define um seção/divisão em um documento (caixa).

span

Define uma seção inline (em linha) ao documento.

section

Define um seção/divisão em um documento.

dialog

Define um caixa
