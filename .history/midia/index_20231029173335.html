Áudio / Vídeo

audio

Usado para adicionar áudios.

source

Usado para definir múltiplos arquivos de mídia para um vídeo ou áudio - ao adicionar duas ou mais opções de arquivo de mídia o navegador escolherá o melhor.
Obs: Precisa especificar o tipo de arquivo de áudio que você está usando, atributo (type).

track

Irá adicionar faixas de texto para o seu elemento de áudio ou vídeo.
