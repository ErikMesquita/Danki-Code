Atributos Globais

accesskey

Especifíca uma tecla de atalho para ativar/ focar um elemento.

class

Especifica um ou mais nomes de classe para um elemento, que poderá ser recuperado posteriormente para estilizar um elemento.

contenteditable

Especifíca se o conteúdo de um elemento é editável ou não.

data-*

Usado para armazenare dados personalizados privados para a página ou aplicativo.

dir

Especifica a direção do texto para o conteúdo em um elemento.

hidden

Oculta um elemento (tag).

id

Especifíca um ID exclusivo para um elemento, que poderá ser recuperado posteriormente para estilizar um elemento.
